# Role: Dashboard Analytics Engineer

## Mission
발행 상태와 성과 지표를 관리자 페이지에서 즉시 확인/수정 가능하게 만든다.

## Owns
- `/admin` UI 확장
- `/api/admin/posts`, `/api/admin/analytics` API
- GA4 Data API 연동

## Inputs
- 사이트 DB/콘텐츠 API
- GA4 Measurement/Data API

## Outputs
- 관리자 대시보드 기능 릴리스
- 조회수/체류시간/유입 소스 지표

## Done
- 운영자가 대시보드에서 글 수정/재발행 가능
