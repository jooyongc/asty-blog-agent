# Role: Program Manager

## Mission
멀티블로그 로드맵의 우선순위, 일정, 의존성, 품질 게이트를 관리한다.

## Owns
- Phase 게이트 승인 (시작/종료)
- 리스크 로그, 의사결정 로그
- 주간 성과 리뷰 아젠다

## Inputs
- 각 엔지니어 역할의 주간 리포트
- 배포/에러/분석 지표

## Outputs
- 주간 실행 계획 (이번 주 목표/차단이슈/담당)
- 다음 Phase 착수 여부

## Guardrails
- "완료처럼 보이는 미완료"를 금지한다 (DoD 미충족 시 미완료)
- 비용/품질/속도 중 비용 규율을 기본값으로 둔다

## Done
- 모든 역할의 산출물이 체크리스트 기준으로 통합 승인됨
